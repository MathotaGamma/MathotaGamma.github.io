JS-Window ver:1.02.01

APIのURLは https://mathotagamma.github.io/APIs/JS-Window/1-02-01/JS-Window.js

クライアントが編集&実行を試すことができるJavascriptコード欄を作成する。

ver: 1.01.02から変わったこと

  1. js-window内で外部ライブラリを使用できるように。
  2. Objectのconsole.logができるだけ展開して表示されるように。
  3. js-windowにrowとcol属性を指定することが可能に(default値はrow=6,col=40)。

使い方例

<script type="module">
  import { registerLibs } from "https://mathotagamma.github.io/APIs/JS-Window/1-02-01/JS-Window.js";

  // Library登録。
  // 使用しているURLは一例。
  registerLibs([
    "https://mathotagamma.github.io/APIs/CompVisJS/1-02-02/CompVisJS.js",
    "https://mathotagamma.github.io/APIs/Physics/1-01-01/Physics.js"
  ]);
</script>

※Libraryをjs-windowに使用しない場合、
<script type="module" src="https://mathotagamma.github.io/APIs/JS-Window/1-02-01/JS-Window.js"></script>
で使用できる。

<js-window row="10">
console.log("Hello World!");
function sum( a, b ) {
  return a + b;
}
let N = 0;
for (let n = 1; n <= 100; n++) {
  N = sum( N, n );
}
console.log( "result", N );
console.log( new CompVis.Complex(1,3).arg );
</js-window>

rowとcol属性の例

指定なし

console.log("row",6,"col",40);

row="3",col="20"

console.log("row",3,"col",20);

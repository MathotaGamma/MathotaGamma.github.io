DQN < Dendraw

APIのURLは(クリックでコピー -->)https://mathotagamma.github.io/API/Dendraw/DQN/latest.js です。

メソッド

net = new DQN(Argument)

Networkを作成する。

Argument

  • bufferSize

    初期値: 1000

  • minBufferSize

    初期値: 0

  • batchSize

    初期値: 32

  • eta

    初期値: 0.005

  • gamma

    初期値: 0.5

  • E

    初期値: (q, d) => 1/2*(q-d)**2

  • dE

    初期値: (q, d) => q-d

  • actionType

    初期値: random

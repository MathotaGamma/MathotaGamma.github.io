JS-Window ver:1.01.01

クライアントが編集&実行を試すことができるJavascriptコード欄を作成する。

例:

<js-window>alert("This is JS-Window");
console.log("Hello World!");
function sum( a, b ) {
return a + b;
}
let N = 0;
for (let n = 1; n <= 100; n++) N = sum( N, n );
console.log( "result", N );
</js-window>
